# anekBot Java

Версия 0.01 beta

Авторы: Ильина Анна, Кихтенко Татьяна

# Описание

Телеграм-бот с математическими шутками

# Доступные /команды

- `/start` — начать общение с ботом
- `/help` — вызов справки
- `/joke` — получить шутку

# Функционал

В первой версии доступно две кнопки: `/joke` и `/help`
